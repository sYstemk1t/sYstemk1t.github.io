2019-08-22 CдёҺжҢҮй’Ҳ c 0 0 иҜ„и®ә еӯ—ж•°з»ҹи®Ў: 2.1k(еӯ—) йҳ…иҜ»ж—¶й•ҝ: 8(еҲҶ)

ж•°з»„гҖҒжҢҮй’Ҳж•ҲзҺҮгҖҒж•°з»„е’ҢжҢҮй’ҲгҖҒдҪңдёәеҮҪж•°еҸӮж•°зҡ„ж•°з»„еҗҚгҖҒеЈ°жҳҺж•°з»„еҸӮж•°гҖҒеҲқе§ӢеҢ–гҖҒдёҚе®Ңж•ҙзҡ„еҲқе§ӢеҢ–гҖҒиҮӘеҠЁи®Ўз®—ж•°з»„й•ҝеәҰгҖҒеӯ—з¬Ұж•°з»„еҲқе§ӢеҢ–гҖҒдәҢз»ҙж•°з»„гҖҒж•°з»„еҗҚгҖҒдёӢж ҮгҖҒжҢҮеҗ‘ж•°з»„зҡ„жҢҮй’ҲгҖҒдҪңдёәеҮҪж•°еҸӮж•°зҡ„еӨҡз»ҙж•°з»„гҖҒжҖ»з»“

ж•°з»„

дёҖз»ҙж•°з»„

жңүж—¶еҖҷжҲ‘д»¬йңҖиҰҒеӨҡдёӘзӣёеҗҢзұ»еһӢзҡ„еҖјпјҢдҪҶжҳҜдҪҝз”Ёз¬¬дёҖз§Қе®ҡд№үж–№ејҸиҝҮдәҺз№ҒзҗҗпјҢжүҖд»ҘжҲ‘д»¬еј•е…ҘдәҶж•°з»„зҡ„жҰӮеҝөпјҢж•°з»„е°ұжҳҜеӨҡдёӘзӣёеҗҢзұ»еһӢзҡ„е…ғзҙ зҡ„еҗҲйӣҶгҖӮ

дҪҶжҳҜиҝҷдёӘж—¶еҖҷbд»ЈиЎЁд»Җд№Ҳпјҹbд»ЈиЎЁзҡ„жҳҜж•ҙдёӘж•°з»„еҗҚпјҢе№¶дё”bжҳҜж•°з»„дёӢж Үзҡ„з¬¬дёҖдёӘе…ғзҙ 

ж•°з»„еҗҚе№¶дёҚз”ЁжҢҮй’ҲеёёйҮҸжқҘиЎЁзӨәвҖ”-ж•°з»„еҗҚеҒҡsizeofж“ҚдҪңз¬ҰжҲ–еј№е№•ж“ҚдҪңз¬Ұ&зҡ„ж“ҚдҪңж•°ж—¶

иҝҷдёӘж—¶еҖҷзҡ„c=bд»ЈиЎЁзҡ„е°ұжҳҜc=bиҝҷдёӘж•°з»„дёӯз¬¬дёҖдёӘе…ғзҙ зҡ„еҖјгҖӮ

иЎЁиҫҫејҸ&b[0]жҳҜдёҖдёӘжҢҮеҗ‘ж•°з»„з¬¬дёҖдёӘе…ғзҙ зҡ„жҢҮй’Ҳ

еҰӮжһңзҗҶи§ЈдәҶ&b[0]жҳҜд»ЈиЎЁйҰ–е…ғзҙ пјҢйӮЈд№Ҳж•°з»„еҗҚд»ЈиЎЁзҡ„д№ҹжҳҜйҰ–е…ғзҙ пјҢйӮЈд№Ҳз¬¬дә”иЎҢд»Јз Ғе°ұеҫҲе®№жҳ“еҸҜд»ҘзҗҶи§ЈдәҶгҖӮ

йӮЈд№Ҳз¬¬е…ӯиЎҢжҳҜзҘһжҒ¶йӯ”ж„ҸжҖқе‘ўпјҢcжҳҜдёҖдёӘжҢҮй’ҲеҸҳйҮҸпјҢaжҳҜдёҖдёӘж•°з»„пјҢжҠҠcиөӢеҖјз»ҷaпјҢиҝҷж ·зҡ„еҶҷжі•жҳҜй”ҷиҜҜзҡ„пјҢaжҳҜдёҖдёӘеёёйҮҸпјҢдёҚеҸҜд»Ҙиў«дҝ®ж”№гҖӮ

дёӢж Үеј•з”Ё

еңЁдҪҝз”ЁдёӢж Үзҡ„ең°ж–№пјҢд№ҹеҸҜд»ҘйҖҡиҝҮеҜ№зӯүзҡ„жҢҮй’ҲиЎЁиҫҫејҸжқҘд»Јжӣҝ

int array[10];

int *ap = array+2;

array+2иҝҷдёӘиЎЁиҫҫејҸд»ЈиЎЁзҡ„жҳҜarray[0]+array[2] = array[2];

еҰӮжһңйңҖиҰҒжҢҮй’Ҳж“ҚдҪңж•°з»„зҡ„ж—¶еҖҷпјҡ

*(ap+(0))иҝҷж ·е°ұжҳҜж“ҚдҪңдәҶиҝҷдёӘж•°з»„зҡ„з¬¬0дёӘе…ғзҙ 

*(array+2)е’ҢдёҠйқўзҡ„иЎЁиҫҫејҸеҜ№зӯү

ap+6,иҝҷдёӘдёӘиЎЁиҫҫејҸе°ұжҳҜзңӢеҪ“еүҚзҡ„apжҢҮеҗ‘ж•°з»„дёӯзҡ„з¬¬еҮ дёӘе…ғзҙ пјҢз„¶еҗҺеҗ‘еҗҺз§»еҮ дёӘе…ғзҙ 

*ap+6пјҢиҝҷдёӘе’ҢдёҠдёҖдёӘдёҚдёҖж ·зҡ„пјҢиҝҷдёӘжңүдёҖдёӘдјҳе…Ҳзә§зҡ„й—®йўҳпјҢиҝҷдёӘиЎЁиҫҫејҸдёӯдјҡе…ҲжүҫеҲ°й—ҙжҺҘеј•з”ЁпјҢз„¶еҗҺз”Ёй—ҙжҺҘеј•з”Ёзҡ„еҖје’Ң6зӣёеҠ 

*(ap+6),жӢ¬еҸ·зҡ„дјҳе…Ҳзә§еӨ§дәҺдёҖеҲҮпјҢжүҖд»Ҙе…Ҳи®Ўз®—жӢ¬еҸ·еҶ…е®№пјҢз„¶еҗҺеҸ–еҶ…е®№

&ap,еҜ№apиҝӣиЎҢеҸ–ең°еқҖ

ap[-1],еҰӮжһңиҝҷдёӘж—¶еҖҷapжҢҮеҗ‘з¬¬дёүдёӘе…ғзҙ пјҢйӮЈд№Ҳ-1пјҢе°ұжҳҜжҢҮеҗ‘з¬¬дәҢдёӘе…ғзҙ пјҢarray[1];

ap[9]пјҢеҪ“еүҚзҡ„apжҢҮеҗ‘зҡ„е…ғзҙ жҳҜ2пјҢдҪҶжҳҜиҝҷдёӘж•°жҚ®еҸӘжңү10дёӘпјҢ9+2зӯүдәҺ11пјҢи®ҝй—®и¶Ҡз•ҢдәҶ

2[array]пјҢиҝҷдёӘиЎЁиҫҫејҸжӢҶи§ЈејҖзҡ„ж„ҸжҖқпјҡ*(array+2)пјҢзӣёеҪ“дәҺдёӢж ҮеҠ 2

дёӢж Үз»қдёҚдјҡжҜ”жҢҮй’Ҳжӣҙжңүж•ҲзҺҮпјҢдҪҶжҳҜжҢҮй’Ҳжңүж—¶еҖҷдјҡжҜ”дёӢж Үжӣҙжңүж•ҲзҺҮ

дёҠйқўз®ҖеҚ•зҡ„е®һзҺ°дёӨз§Қж–№жі•пјҢжҠҠдёҖдёӘж•°з»„еҶ…зҡ„жүҖжңүеҶ…е®№е…ЁйғЁеҪ’дёә0

жҢҮй’Ҳж•ҲзҺҮ

жҢҮй’Ҳжңүж—¶еҖҷжҜ”дёӢж Үжӣҙжңүж•ҲзҺҮпјҢеүҚжҸҗжҳҜд»–д»¬еҫ—иў«жӯЈзЎ®зҡ„дҪҝз”ЁгҖӮ

ж•°з»„е’ҢжҢҮй’Ҳе№¶дёҚжҳҜзӣёзӯүзҡ„

еЈ°жҳҺдёҖдёӘж•°з»„зҡ„ж—¶еҖҷпјҢзј–иҜ‘еҷЁдјҡж №жҚ®ж•°з»„зҡ„еӨ§е°Ҹд»ҘеҸҠзұ»еһӢдёәе®ғејҖиҫҹдёҖеқ—еҶ…еӯҳз©әй—ҙгҖӮ

еЈ°жҳҺдёҖдёӘжҢҮй’Ҳзҡ„ж—¶еҖҷпјҢзј–иҜ‘еҷЁеҸӘдёәжҢҮй’Ҳжң¬иә«дҝқз•ҷеҶ…еӯҳз©әй—ҙпјҢе®ғе№¶дёҚдёәд»»дҪ•ж•°жҚ®зұ»еһӢеҲҶй…ҚеҶ…еӯҳз©әй—ҙ

дҪңдёәеҮҪж•°еҸӮж•°зҡ„ж•°з»„еҗҚ

еҰӮжһңжҠҠдёҖдёӘж•°з»„еҗҚеҒҡеҸӮж•°дј йҖ’з»ҷеҮҪж•°пјҢи°ғз”ЁеҮҪж•°е®һйҷ…дёҠдј йҖ’зҡ„жҳҜдёҖдёӘжҢҮй’ҲпјҢжүҖд»ҘеҮҪж•°зҡ„еҪўеҸӮе®һйҷ…дёҠжҳҜдёӘжҢҮй’Ҳ

и®Іеӯ—з¬ҰдёІbдёӯзҡ„еҶ…е®№жӢ·иҙқеҲ°еӯ—з¬ҰдёІaдёӯпјҢиҷҪз„¶cиҜӯиЁҖжҸҗдҫӣдәҶеҫҲеӨҡеә“еҮҪж•°

еЈ°жҳҺж•°з»„еҸӮж•°

и°ғз”ЁеҮҪж•°ж—¶е®һйҷ…дј йҖ’зҡ„жҳҜдёҖдёӘжҢҮй’ҲпјҢжүҖд»ҘеҮҪж•°зҡ„еҪўеҸӮжҳҜдёӘжҢҮй’Ҳ

дҪҶжҳҜзј–иҜ‘д№ҹжҺҘеҸ—ж•°з»„еҪўејҸзҡ„еҮҪж•°еҪўеҸӮ

иҝҷдёӘд»Јз ҒжңҖеҗҺжү“еҚ°зҡ„ж—¶еҖҷеҖје№¶дёҚжҳҜ1пјҢдҪҶжҳҜеүҚйқўиҜҙдәҶпјҢж•°жҚ®еҗҚзӣёеҪ“дәҺз¬¬дёҖдёӘе…ғзҙ пјҢдҪҶжҳҜиҝҷйҮҢжү“еҚ°еҮәжқҘзҡ„жҳҜдёҖдёӘж•°з»„пјҢиҝҷйҮҢе°ұжңүдёҖдёӘй—®йўҳе°ұжҳҜпјҢеҰӮжһңд»Ҙж•ҙеҪўзҡ„ж–№ејҸзҡ„жү“еҚ°ж•°з»„пјҢе°ұдјҡеҮәй—®йўҳпјҢж јејҸеҢ–иҫ“еҮәзҡ„еҪўејҸжҳҜж•ҙеҪўпјҢдҪҶжҳҜжҲ‘д»¬зҡ„зұ»еһӢжҳҜдёҖдёӘж•°з»„гҖӮ
