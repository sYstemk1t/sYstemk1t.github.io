Windows SDkж–ҮжЎЈеЈ°жҳҺпјҢиҝӣзЁӢиҰҒзӯүеҲ°жүҖжңүзәҝзЁӢз»ҲжӯўиҝҗиЎҢд»ҘеҗҺжүҚз»ҲжӯўиҝҗиЎҢгҖӮеҰӮжһңеңЁе…ҘеҸЈзӮ№и°ғз”Ёзҡ„ExitThreadпјҢиҖҢдёҚжҳҜи°ғз”ЁExitProcessжҲ–иҖ…д»…д»…жҳҜиҝ”еӣһпјҢйӮЈд№Ҳеә”з”ЁзЁӢеәҸзҡ„дё»зәҝзЁӢе°ҶеҒңжӯўиҝҗиЎҢпјҢдҪҶжҳҜеҰӮжһңиҝӣзЁӢдёӯиҮіе°‘иҝҳжңүдёҖдёӘзәҝзЁӢеңЁиҝҗиЎҢпјҢиҜҘиҝӣзЁӢдёҚдјҡз»ҲжӯўиҝҗиЎҢгҖӮ

е…ЁеұҖеҜ№иұЎе’ҢеұҖйғЁеҜ№иұЎйғҪжІЎжңүи°ғз”Ёжһҗжһ„еҮҪж•°пјҢC++еҜ№иұЎжІЎжңүиў«жӯЈзЎ®жһҗжһ„пјҢеӣ дёәExitProcessйҖ жҲҗиҝӣзЁӢеҪ“еңәз»ҲжӯўиҝҗиЎҢгҖӮC/C++иҝҗиЎҢеә“жІЎжңүжңәдјҡжү§иЎҢжё…зҗҶе·ҘдҪңгҖӮ

еҸӘйңҖиҰҒдё»зәҝзЁӢзҡ„е…ҘеҸЈеҮҪж•°иҝ”еӣһпјҢC/C++иҝҗиЎҢеә“е°ұиғҪжү§иЎҢе…¶жё…зҗҶе·ҘдҪңгҖӮ

TerminateProcess

и°ғз”ЁTerminateProcessд№ҹеҸҜд»Ҙз»ҲжӯўдёҖдёӘиҝӣзЁӢ

д»»дҪ•зәҝзЁӢйғҪеҸҜд»Ҙи°ғз”ЁTerminateProcessжқҘз»ҲжӯўеҸҰдёҖдёӘиҝӣзЁӢжҲ–иҖ…иҮӘе·ұзҡ„иҝӣзЁӢгҖӮ hProcessжҢҮе®ҡдәҶиҰҒз»Ҳжӯўзҡ„иҝӣзЁӢзҡ„еҸҘжҹ„гҖӮе…¶йҖҖеҮәд»Јз Ғзҡ„еҖје°ұжҳҜдј з»ҷuExitCodeзҡ„еҖјгҖӮ

ж“ҚдҪңзі»з»ҹеңЁиҝӣзЁӢз»Ҳжӯўд»ҘеҗҺдјҡиҝӣиЎҢеҪ»еә•жё…зҗҶпјҢдҝқиҜҒдёҚдјҡжі„йңІд»»дҪ•ж“ҚдҪңзі»з»ҹиө„жәҗгҖӮеңЁиҝӣзЁӢз»ҲжӯўеҗҺз»қеҜ№дёҚдјҡжі„жјҸд»»дҪ•дёңиҘҝгҖӮ

TerminateProcessеҮҪж•°жҳҜејӮжӯҘзҡ„пјҢе№¶дёҚзӯүеҲ°иҝӣзЁӢе®Ңе…Ёз»ҲжӯўдәҶжүҚиҝ”еӣһгҖӮдёәдәҶзЎ®е®ҡиҝӣзЁӢжҳҜеҗҰз»ҲжӯўйңҖиҰҒдҪҝз”ЁWaitForSingleObject

еҪ“иҝӣзЁӢдёӯзҡ„жүҖжңүзәҝзЁӢз»Ҳжӯўж—¶

еҪ“иҝӣзЁӢзҡ„жүҖжңүзәҝзЁӢз»Ҳжӯўж—¶пјҢж“ҚдҪңзі»з»ҹи®ӨдёәжІЎжңүд»»дҪ•зҗҶз”ұеҶҚдҝқжҢҒиҝӣзЁӢзҡ„ең°еқҖз©әй—ҙгҖӮе№¶дјҡз»ҲжӯўиҝҷдёӘиҝӣзЁӢгҖӮиҝӣзЁӢзҡ„йҖҖеҮәд»Јз Ғдјҡиў«и®ҫзҪ®дёәжңҖеҗҺдёҖдёӘз»Ҳжӯўзҡ„е“ӘдёӘзәҝзЁӢзҡ„йҖҖеҮәд»Јз ҒгҖӮ

еҪ“иҝӣзЁӢз»ҲжӯўиҝҗиЎҢж—¶

дёҖдёӘиҝӣзЁӢз»Ҳжӯўж—¶пјҢзі»з»ҹеӣһдҫқж¬Ўжү§иЎҢд»ҘдёӢж“ҚдҪңгҖӮ

  • з»ҲжӯўиҝӣзЁӢдёӯй©»з•ҷзҡ„д»»дҪ•зәҝзЁӢ
  • йҮҠж”ҫиҝӣзЁӢеҲҶй…Қзҡ„жүҖжңүз”ЁжҲ·еҜ№иұЎе’ҢGDIеҜ№иұЎпјҢе…ій—ӯжүҖжңүеҶ…ж ёеҜ№иұЎпјҲеҰӮжһңжІЎжңүе…¶д»–иҝӣзЁӢжү“ејҖиҝҷдәӣеҶ…ж ёеҜ№иұЎзҡ„еҸҘжҹ„пјҢйӮЈд№Ҳд»–д»¬дјҡиў«й”ҖжҜҒгҖӮеҗҰеҲҷеј•з”Ёи®Ўж•°еҷЁ-1пјү
  • иҝӣзЁӢзҡ„йҖҖеҮәд»Јз Ғд»ҺSTILL_ACTIVEеҸҳдёәдј з»ҷExitProcessжҲ–TerminateProcessеҮҪж•°зҡ„д»Јз Ғ
  • иҝӣзЁӢеҶ…ж ёеҜ№иұЎзҠ¶жҖҒеҸҳдёәе·Іи§ҰеҸ‘зҠ¶жҖҒ
  • иҝӣзЁӢеҶ…ж ёеҜ№иұЎеј•з”Ёи®Ўж•°еҷЁ-1

еҪ“зҲ¶иҝӣзЁӢеҝҳи®°е…ій—ӯеӯҗиҝӣзЁӢеҸҘжҹ„ж—¶еҖҷпјҢеӯҗиҝӣзЁӢеҚідҪҝз»“жқҹгҖӮе…¶иҝӣзЁӢеҶ…ж ёеҜ№иұЎеҸҘжҹ„д»Қз„¶дёҚдјҡиў«й”ҖжҜҒгҖӮжӯӨж—¶еҸҜд»ҘиҺ·еҫ—дёҖдәӣз»ҹи®ЎдҝЎжҒҜпјҢдҫӢеҰӮGetExitCodeProcessжқҘиҺ·еҫ—дёҖдёӘе·Із»Ҹз»Ҳжӯўзҡ„иҝӣзЁӢзҡ„йҖҖеҮәд»Јз ҒгҖӮ

еҰӮжһңиў«и°ғз”Ёзҡ„иҝӣзЁӢжңӘз»ҲжӯўпјҢlpExitCodeдјҡиҝ”еӣһ STILL_ACTIVE(0x103)

йҮҚзӮ№пјҢеҰӮжһңеҜ№иҝӣзЁӢзҡ„з»ҹи®Ўж•°жҚ®дёҚеҶҚж„ҹе…ҙи¶Јеә”иҜҘи°ғз”ЁCloseHandleжқҘйҖ’еҮҸеҶ…ж ёеҜ№иұЎзҡ„дҪҝз”Ёи®Ўж•°еҷЁпјҢе№¶йҮҠж”ҫе®ғгҖӮ

еӯҗиҝӣзЁӢ

дёҖдёӘиүҜеҘҪзҡ„зј–зЁӢд№ жғҜпјҡеңЁдёҚйңҖиҰҒдҪҝз”ЁеӯҗиҝӣзЁӢзҡ„зӣёе…іеҶ…ж ёеҜ№иұЎеә”иҜҘз«ӢеҚіCloseHandleгҖӮ еҗҰеҲҷеҒҮе®ҡеӯҗиҝӣзЁӢз”ҹжҲҗдәҶеҸҰеӨ–дёҖдёӘж–°зҡ„еӯҗиҝӣзЁӢпјҢиҖҢиҮӘе·ұзҡ„дё»зәҝзЁӢе·Із»ҸйҖҖеҮәгҖӮз”ұдәҺи°ғз”ЁCreateProcessзҡ„еҪ“еүҚиҝӣзЁӢжңӘйҮҠж”ҫеӯҗиҝӣзЁӢзҡ„иҝӣзЁӢеҜ№иұЎпјҢеӣ жӯӨиҮӘиҝӣзЁӢзҡ„еҶ…ж ёеҜ№иұЎдёҚдјҡиў«ж“ҚдҪңзі»з»ҹйҮҠж”ҫгҖӮ

иҝҗиЎҢзӢ¬з«Ӣзҡ„еӯҗиҝӣзЁӢ

еӨ§еӨҡж•°ж—¶еҖҷеә”з”ЁзЁӢеәҸе°ҶеҸҰдёҖдёӘиҝӣзЁӢдҪңдёәзӢ¬з«Ӣзҡ„иҝӣзЁӢпјҲdetached processпјүжқҘеҗҜеҠЁгҖӮиҝҷе°ұж„Ҹе‘ізқҖдёҖж—ҰеӯҗиҝӣзЁӢеҲӣе»әпјҢзҲ¶иҝӣзЁӢе°ұдёҚеҶҚдёҺе…¶йҖҡдҝЎпјҢжҲ–иҖ…дёҚеҝ…зӯүд»–е®ғе®ҢжҲҗе·ҘдҪңд№ӢеҗҺеҶҚз»§з»ӯиҮӘе·ұзҡ„е·ҘдҪңпјҲеҪ“еүҚиҝӣзЁӢдёҚеҝ…жҢӮиө·зӯүеҫ…пјүгҖӮиҝҷж—¶еҖҷеҸӘйңҖиҰҒи°ғз”ЁCloseHandleе…ій—ӯеӯҗиҝӣзЁӢзҡ„иҝӣзЁӢеҸҘжҹ„е’Ңдё»зәҝзЁӢеҸҘжҹ„гҖӮ
