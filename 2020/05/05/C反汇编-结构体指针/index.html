C反汇编 - 结构体指针

2020-05-05 反汇编 Assembly 0 0 评论 字数统计: 756(字) 阅读时长: 3(分)

结构体指针，可细分为指向结构体变量的指针和指向结构体数组的指针。

结构体指针

结构体指针

定义

“结构体变量名.成员名”的方式引用结构体变量中的成员，除了这种方法之外还可以使用指针

指针自增

结构体指针做加法，一定要考虑结构体的对齐，做加法的时候，去掉指针的修饰符号，然后结构体多大，就加多少

指针自减

减去一个”*”号，然后再减去结构体的大小

成员赋值

结构体指针赋值通过”->”，结构体赋值通过”.”出来

类型强转

让指针做加法

任何指针类型都不需要管是什么类型给他赋值
