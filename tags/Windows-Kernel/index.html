Windows核心编程 - 进程

进程是正在运行的程序的一个实例，它由两部分组成：

  • 一个内核对象，操作系统用它来管理进程；
  • 一个地址空间，其中包含所有可执行文件或DLL模块的代码和数据；

对于进程的调度，所有要运行的现程，操作系统会轮流为每个现程调度一些CPU事件。它会采用轮盘的方式，为每个现程都分配时间片，从而营造出所有现程都在”并发”的假象。
